CSS Selectors

1. Descendant Selector

Selects all elements that are descendants (children, grandchildren, etc.) of a specified ancestor.

  • Syntax: ancestor descendant (e.g., ol li)
  • Example: 
    ol li {
    color: blue;
}
    This styles all <li> inside an <ol>, including nested lists.

2. Child Selector

Selects only immediate children of a parent element.

  • Syntax: parent > child
  • Example: 
    ol > li {
    color: red;
}
    Only the top-level <li> inside <ol> are styled; nested items are ignored.

3. Attribute Selector

Selects elements based on attributes and their values.

  • Syntax: element[attribute="value"]
  • Example: 
    a[href="https://facebook.com"] {
    color: blue;
}
    Applies styles only to <a> tags with that exact href value.

4. Pseudo-Class Selector

Selects elements based on state or user interaction rather than document structure.

  • Syntax: element:pseudo-class
  • Example: 
    button:hover {
    background-color: orange;
}
    Changes button color when the user hovers over it.
