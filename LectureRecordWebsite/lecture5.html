Lecture 5

Introduction to HTML5

1. What is HTML5?

HTML5 is the latest version of the HTML standard designed for modern web development. It introduces new features that improve webpage structure, multimedia support, and device compatibility.

Key Focus Areas

  • Better semantic structure
  • Native multimedia support
  • Device independence and responsive design
HTML5 Doctype: <!DOCTYPE html>

2. Key Features of HTML5

  • Responsive Design: Websites adapt to different screen sizes using the viewport meta tag.
  • Native Multimedia: Supports <audio> and <video> elements without plugins like Flash.
  • Cross-Platform Compatibility: Features like <canvas> allow interactive graphics.
  • New Form Input Types: Includes email, tel, number, and date inputs.
  • Offline Support: Technologies like localStorage and service workers allow web apps to work offline.

3. Importance of Semantic HTML

Semantic HTML elements clearly describe the meaning of the content to both browsers and developers.

Benefits

  • Improves accessibility for assistive technologies
  • Enhances SEO by helping search engines understand content
  • Makes code easier to read and maintain

4. Semantic vs Non-Semantic Elements

Non-Semantic Elements

  • <div>
  • <span>
  • Do not describe the meaning of their content

Semantic Elements

  • <form>
  • <table>
  • <article>
  • <nav>
  • Clearly define the purpose of the content

5. Key Semantic Elements in HTML5

  • <header> – Introductory content such as logos or headings
  • <footer> – Footer section containing copyright or links
  • <article> – Independent content such as blog posts
  • <section> – Groups related content
  • <nav> – Contains navigation links
  • <aside> – Sidebar or related content
  • <figure> & <figcaption> – Used for images with captions

6. Multimedia and Graphics

  • <audio> – Embeds audio files in a webpage
  • <video> – Embeds video content with built-in controls
  • <canvas> – Used to draw graphics, animations, and games with JavaScript
HTML5 allows multimedia without requiring external plugins.

7. HTML5 Storage Options

  • localStorage: Stores data with no expiration time.
  • sessionStorage: Stores data only for the current session.
These storage methods allow web applications to store data locally in the browser.
