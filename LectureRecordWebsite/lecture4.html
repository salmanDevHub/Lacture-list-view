Lecture 4

Tables, Div, Span, and Layout Design in HTML

1. HTML Tables (<table>)

HTML tables are used to display data in rows and columns. They are useful when information needs to be presented in a structured format.

Basic Structure

  • <table> – Defines a table
  • <tr> – Table row
  • <th> – Table header
  • <td> – Table data

Important Attributes

  • border – Adds border around table
  • cellpadding – Space inside cells
  • cellspacing – Space between cells

Merging Cells

  • colspan – Merge columns
  • rowspan – Merge rows
Important: Tables should only be used for displaying data, not for page layout design.

2. The Div Element (<div>)

The <div> element is a block-level container used to group larger sections of content on a webpage.

  • Creates a new line
  • Takes full width of the page
  • Used to divide webpage sections such as header, content, and footer
  • Helps in building flexible layouts
Example use: Page header, navigation bar, main content, sidebar, footer

3. The Span Element (<span>)

The <span> element is an inline container used to style small parts of text inside a paragraph or heading.

  • Does not start on a new line
  • Used to apply style to a small text portion
  • Works well with CSS styling
Example: Highlighting a word inside a paragraph using CSS.

4. Div vs Span: Key Difference

Div (Block-Level)

  • Starts on a new line
  • Takes full width available
  • Used for grouping larger sections of content

Span (Inline)

  • Stays on the same line
  • Takes only the required width
  • Used for styling small text sections

5. Modern Layout Best Practices

  • Use <div> for page structure and layout sections
  • Use <span> for styling inline text
  • Avoid using tables for webpage layout
  • Use tables only for displaying tabular data
  • Nesting multiple <div> elements helps build complex layouts
Best Practice: Combine HTML structure with CSS (Flexbox or Grid) to create responsive and modern webpage layouts.
