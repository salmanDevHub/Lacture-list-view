Lecture 6

HTML Forms

1. Introduction to HTML Forms

HTML forms are used to collect user input on websites such as login pages, registration forms, surveys, and feedback forms.

  • Forms capture data from users
  • Data is sent to a server for processing or storage
  • Common uses include login, signup, and contact forms

2. The <form> Element

The <form> element acts as a container for all input fields and controls used in a form.

Important Attributes

  • action: URL where the form data will be sent.
  • method: Defines how data is sent to the server.
    • GET – Data is appended to the URL
    • POST – Data is sent inside the request body
  • target: Specifies where the response will appear (e.g., _blank opens in a new tab).

3. Core Form Elements

<label>

  • Associates text with a form control
  • Clicking the label focuses the related input field

<input>

  • Most commonly used form element
  • Behavior depends on the type attribute
  • Common types: text, password, email, checkbox, radio

<select> and <option>

  • Create dropdown lists
  • <option> defines items inside the dropdown

<textarea>

  • Used for multi-line text input
  • Commonly used for comments or messages

<button>

  • Used to submit or reset forms
  • Can also trigger JavaScript actions

4. Key Form Attributes

  • name: Identifies the form field when data is sent to the server.
  • value: Sets the default value for an input field.
  • placeholder: Displays hint text inside the input field.

Validation Attributes

  • required: Ensures the field must be filled before submission.
  • maxlength: Limits the number of characters.
  • pattern: Validates input using a regular expression.

5. Form Submission Process

When a user submits a form, the browser collects all data from the input fields.

  • Data is collected from each field
  • Fields are sent as key-value pairs
  • Key = name attribute
  • Value = user input
  • Data is sent to the URL specified in the action attribute
  • The method (GET or POST) determines how the data is transmitted
Example format: name=Salman & age=20
