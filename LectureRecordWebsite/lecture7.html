Lecture 7

Introduction to CSS

1. What is CSS?

CSS (Cascading Style Sheets) is the language used to style webpages. It controls how HTML elements appear on the screen.

  • Defines colors, layout, fonts, and spacing
  • Allows styling of multiple pages using one stylesheet
  • External styles are stored in .css files

2. CSS Syntax

A CSS rule contains a selector and a declaration block.

Example: 
h1 {
  color: red;
  text-align: right;
}
Explanation:
  • h1 → selector
  • color → property
  • red → value

3. Ways to Use CSS

  • Inline CSS Applied directly inside an HTML element using the style attribute.
  • Internal CSS Defined inside a <style> tag within the HTML <head>.
  • External CSS Stored in a separate .css file and linked to HTML (best practice).

4. Typeface vs Font

  • Typeface: The design of characters (e.g., Arial, Helvetica).
  • Font: A specific size, weight, or style of a typeface.

Common Font Families

  • Serif – Traditional style with strokes (Times New Roman)
  • Sans-serif – Clean modern fonts (Arial, Helvetica)
  • Monospace – Same width characters (Courier New)
  • Cursive – Handwriting style fonts
  • Fantasy – Decorative fonts

5. The CSS Box Model

Every HTML element is treated as a rectangular box.

  • Content – The text or image inside the element
  • Padding – Space between content and border
  • Border – Edge surrounding the padding
  • Margin – Space outside the border separating elements
Content → Padding → Border → Margin

6. Identifying Elements: ID vs Class

  • ID (#) Used to uniquely identify one element on a page.
  • Class (.) Can be reused for multiple elements.
Example CSS: 
#header{
background:blue;
}

.button{
color:white;
}

7. CSS Specificity (Precedence)

Specificity determines which CSS rule is applied when multiple rules target the same element.

Order of Priority

  • Inline Styles
  • ID Selectors
  • Class Selectors
  • Element Selectors
Example: An ID selector will override a type selector.
