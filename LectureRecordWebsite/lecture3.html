Introduction to HTML

1. What is HTML?

HTML (HyperText Markup Language) is the standard language used to create webpages. It defines the structure of a webpage using elements and tags.

Key Concepts

  • HyperText: Text that contains links to other documents.
  • Markup Language: A language that uses tags to define elements inside a document.

2. History & Evolution of HTML

  • 1989: Tim Berners-Lee invented HTML at CERN.
  • 1993 – 1997: HTML 1.0, 2.0 and 4.0 were introduced with features like forms and tables.
  • 2014: HTML5 became the current standard with audio/video support and semantic tags.
Key Feature of HTML5: Native support for multimedia such as audio and video without external plugins.

3. XML vs XHTML

XML (Extensible Markup Language)

  • Used for storing and transporting data.
  • Allows custom tags such as <student>.
  • Strict syntax rules and case-sensitive.
  • All tags must be properly closed.

XHTML (Extensible HyperText Markup Language)

  • A stricter version of HTML based on XML.
  • All tags must be written in lowercase.
  • Elements must be properly nested.
  • Every tag must be closed.

4. HTML vs XHTML: Key Difference

  • HTML: Flexible syntax and allows some tags without closing.
  • XHTML: Strict syntax rules and requires all tags to be closed.
Example:
HTML: <img>
XHTML: <img />

5. HTML Elements & Tags

  • HTML Element: Combination of opening tag, content, and closing tag.
  • Attributes: Provide extra information about elements.

Common HTML Tags

  • <h1> to <h6> – Headings
  • <p> – Paragraph
  • <a> – Links
  • <img> – Images
  • <ul> / <ol> – Lists

6. Block-Level vs Inline Elements

Block-Level Elements

  • Start on a new line
  • Take full width available
  • Examples: <div>, <p>, <h1>, <ul>

Inline Elements

  • Do not start on a new line
  • Take only required width
  • Examples: <span>, <a>, <img>, <strong>

7. HTML Validation

HTML validation checks whether the HTML code follows proper syntax and W3C standards.

Importance of Validation

  • Ensures compatibility across browsers
  • Helps detect coding errors
  • Improves SEO performance
  • Ensures accessibility for users with disabilities
Validation Tool: W3C Markup Validator
https://validator.w3.org
